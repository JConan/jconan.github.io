## � Quick Start

1. **Clone and Setup**

   ```bash
   git clone https://github.com/jconan/jconan.github.io.git
   cd jconan.github.io
   pnpm install
   ```

2. **Development**

   ```bash
   pnpm dev
   ```

3. **Build for Production**
   ```bash
   pnpm build
   ```
